Welcome to my little site.

Select 'Poses' to view a list of common yoga poses.

Each pose will display its English and corresponding Sanskrit name, a simple visualization, and a list of categories related to that pose.

Select 'Categories' to view a list of basic yoga categories.

Select 'Favorites' to view all poses that have been favorited on the 'Poses' page.

Select 'Add Routine' to complete the form to create a new routine.

Fill out the routine title, creator name, difficulty, and included poses to submit a new routine.

Select 'View Routines' to see a list of routines that have been submitted to date.

Routines are listed in order by date posted, oldest to newest.
